# Business Problem

Credit risk and default prediction are fundamental challenges in financial services, where institutions must balance portfolio growth, profitability, and risk exposure.

This thesis explored how predictive analytics and machine learning methodologies can support more informed decision-making by identifying patterns associated with customer risk profiles and potential default behavior.

The project focused on designing a structured analytical framework capable of:

* Identifying relevant predictive variables
* Structuring data for supervised modeling
* Comparing alternative predictive approaches
* Evaluating model performance
* Supporting explainable segmentation and decision frameworks

# Key Takeaways

This project highlighted several practical realities of applying Machine Learning to credit risk classification in a real financial context:

* The predictive value of a model depends heavily on data preparation, variable quality, and problem framing, often more than on algorithmic sophistication alone.
* In credit-related use cases, model interpretability is not optional; business adoption depends on being able to justify outcomes in a structured and understandable way.
* Feature engineering and segmentation decisions are strongly influenced by operational and commercial context, not just statistical performance.
* Building an effective predictive framework requires balancing model performance with stability, usability, and decision-making relevance.
* Real-world financial modeling extends beyond technical implementation and requires integrating analytical rigor with business constraints, governance, and practical applicability.
